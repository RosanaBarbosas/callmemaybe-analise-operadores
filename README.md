Projeto de Análise de Dados — CallMeMaybe

Este projeto tem como objetivo identificar operadores ineficientes no serviço de telefonia virtual CallMeMaybe, apoiando a tomada de decisão por parte de supervisores e gestores. Um operador é considerado ineficiente quando apresenta desempenho inferior em indicadores relacionados à qualidade do atendimento, como alta taxa de chamadas perdidas e tempo médio de espera elevado.

A análise foi conduzida a partir de dados operacionais de chamadas recebidas e realizadas por operadores, utilizando técnicas de análise exploratória de dados (EDA), criação de indicadores-chave (KPIs) e testes estatísticos para validação de hipóteses. Os principais KPIs analisados incluem a taxa de chamadas perdidas, o tempo médio de espera e o volume de chamadas de saída.

Como resultado, foi desenvolvido um dashboard interativo no Tableau Public, que apresenta rankings dos operadores com pior desempenho nos principais indicadores, permitindo a identificação rápida de gargalos operacionais e subsidiando ações corretivas, como treinamentos direcionados e ajustes nos processos de atendimento.

O projeto contempla três entregáveis principais:

Notebook Jupyter, com todo o processo analítico documentado;

Dashboard interativo, voltado à visualização e comunicação dos resultados;

Relatório final em PDF, estruturado no formato CAR (Context, Actions, Results), com foco nos impactos para o negócio.
